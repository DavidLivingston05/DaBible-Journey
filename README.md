# 📖 DaBible-Journey (Scripture Journey)

A premium, modern Bible reading tracker designed for congregations and individuals to monitor their reading progress, share daily reflections, and participate in Scripture trivia.

---

## ✨ Features

- **📊 Comprehensive Progress Dashboard**: Real-time stats showcasing chapters read, Old Testament vs. New Testament progress, and group averages.
- **👥 Congregation Profiles**: Track reading progress across the entire community, sorted dynamically by streaks and chapters read.
- **💡 Reflections & Thoughts**: Share insights, notes, and encouragement from daily scripture readings.
- **🏆 Daily Scripture Trivia**: Engage users with interactive daily challenges to test and grow their knowledge of the Bible.
- **📱 Responsive, Sleek Design**: Premium dark and light modes, custom gradients, fluid animations, and mobile-friendly bottom navigation.
- **📤 Share Progress**: Generate downloadable progress cards or share stats directly via WhatsApp.

---

## 🚀 Running Locally

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```
   Open your browser and navigate to `http://localhost:3000`.

---

## ⚡ Vercel Deployment

This project is fully optimized for hosting on **Vercel** with client-side SPA routing support.

1. Import this repository into Vercel.
2. Vercel will auto-detect the framework (Vite) and configuration.
3. Once deployed, remember to add your Vercel deployment URL to the **Authorized Domains** list under the **Authentication** -> **Settings** tab in your Firebase Console.
